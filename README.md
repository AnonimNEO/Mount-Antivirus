# Подробней о проекте Антивирус Монтировка
Антивирус Монтировка - является open-source проектом на Python3, распространяющимся по лицензии GPL v3.0
Что-то мини антивируса и набора утилит для обнаружения и ликвидации вирусов с ОС Windows. Проект открыт и каждый может поучаствовать в разработке, пусть жизнь в ПК станет безопасней.
Если вы считаете, что это проект бесполезен или он сделан через жопу, то поставте Linux и не партесь об этом.
